# 📦 Sistema de Cadastro de Produtos

Sistema CRUD desenvolvido em Python utilizando SQLite para gerenciamento de produtos via terminal.

---

## 🚀 Funcionalidades

✔️ Cadastrar produtos  
✔️ Listar produtos  
✔️ Buscar produtos  
✔️ Editar produtos  
✔️ Excluir produtos  
✔️ Armazenamento em banco de dados SQLite
